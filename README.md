# COVID--19-tracker
  The idea of this application is to provide a simple GUI application to users to get the track covid cases of any district in India they wish to see. The system also provides a simple user interface for simplification of application. It also provides an amazing UX for its users. The features of this application will be that this will be a real-time tracker app that returns the active, confirmed, deceased and recovered of the searched district. It can also change its theme according to the time of day.
